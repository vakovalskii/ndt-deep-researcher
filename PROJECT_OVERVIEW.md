# Deep Research Assistant - Обзор проекта

## Описание
Мультиязычный исследовательский ассистент на базе LangGraph и LangSmith SDK со структурированным выводом и итеративным интернет-поиском через Tavily API.

## Архитектура
- **LangGraph**: Основной движок для обработки запросов и управления состоянием
- **Tavily API**: Поисковый сервис для получения актуальной информации из интернета
- **LangSmith Studio**: Web-интерфейс для взаимодействия с ассистентом
- **vLLM**: Удалённый сервер для обработки LLM запросов

## Основные компоненты

### 1. Configuration (src/assistant/configuration.py)
- Управление конфигурацией через переменные окружения
- Автоматическая загрузка `.env` файлов
- Валидация настроек при инициализации

### 2. Graph (src/assistant/graph.py)
- Основной граф обработки запросов
- Цикл исследования с максимум 3 итерациями
- Структурированный вывод через JSON схемы

### 3. State (src/assistant/state.py)
- Управление состоянием между этапами обработки
- Накопление результатов поиска
- Отслеживание языка взаимодействия

### 4. Utils (src/assistant/utils.py)
- Функции для работы с поисковыми API (Tavily, Perplexity)
- Форматирование и дедупликация источников
- Ограничение токенов

## Процесс работы
1. **detect_language** - Определение языка входного запроса
2. **generate_query** - Генерация поискового запроса
3. **web_research** - Поиск через Tavily API
4. **summarize_sources** - Суммаризация найденной информации
5. **reflect_on_summary** - Анализ пробелов и генерация следующего запроса
6. **finalize_summary** - Формирование итогового отчёта

## Ключевые особенности
- Поддержка множественных языков
- Итеративный поиск с улучшением запросов
- Структурированный вывод в JSON формате
- Автоматическое управление токенами
- Веб-интерфейс для удобного взаимодействия

## Ограничения
- Максимум 3 итерации исследования
- 2000 токенов на ответ LLM
- 8000 токенов в результатах поиска
- Зависимость от внешних API (Tavily, vLLM)
